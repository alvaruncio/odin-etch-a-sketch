# Etch-a-Sketch

Browser sketchpad built with HTML, CSS and JavaScript as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

A grid of square divs (16×16 by default) drawn using Flexbox. Hover over squares to "draw" by changing their background color. Click the button to reset the grid with a custom size (up to 100×100).

## Features

- 16×16 default grid (960px container)
- Draw by hovering with the mouse
- Button to create a new grid (prompt with validation, max 100)
- Dynamic square sizing (960 / squares per side)
